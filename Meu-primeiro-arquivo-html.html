O que é HTML?

O HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

O HTML ajuda na construção de sites, onde você precisará conhecer a linguagem HTML, acrescentando códigos tags, que indicarão parágrafos, quebra de texto, espaçamento, imagens, entre outros.

Para utilizar o HTML você precisa de um editor de texto, como por exemplo visual code, notepadd++ entre outros, que vão premitir que você crie seus projetos com mais facilidade.

Inspetor de elementos

Ferramenta que permite você explore como os sites foram criados e configurados.

Para acessar essa ferramenta basta apertar F5 ou clicar com o botão direito do mouse e selecionar "inspecionar elemento", abrirá uma aba suspensa que mostra todas as configurações usadas no site.

Essa ferramenta também permite que você faça alterações na página que está verificando (essas modificações não alteram a página em si,apenas te permite observar como alterar alguns padrões).

O que são Tags?

As tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

As tags HTML seguem todas o mesmo padrão, são usadoas da mesma maneira em todos os lugares.

As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas. O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere.

Algumas tags para quem está começando a usar HTML:
  • read_ faz as definições que não ficam visíveis para o usuário
  • body_ é o que define o corpo do arquivo HTML
  • p_ cria parágrafos
  • H1/H2 ... _ usadas para títulos, o tamanho da fonte diminui de acordo com o número usado
  • i_ coloca o testo em italico
  • u_ sublinha o texto
  • strong_ deixa o texto em negrito
  • ol_ cria listas ordenadas1
  • ul-cria listas não ordenadas2
